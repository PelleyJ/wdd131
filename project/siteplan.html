Motorcycle Mania - Site Plan

Your hub for motorcycle passion and exploration

Site Name

Motorcycle Mania

This name reflects the excitement and energy surrounding motorcycles, from beginner riders to enthusiasts exploring gear, routes, and events. Optional domain availability: motorcyclemania.org

Site Purpose

This website provides useful resources for motorcycle enthusiasts. It includes guides for new riders, gear recommendations, scenic ride ideas, upcoming motorcycle events, and a contact form for questions or club membership.

Scenarios

  • What is the best helmet for long-distance touring?
  • Where are the top 5 motorcycle routes in Japan?

Color Schema

  • Primary Color: #1a1a1a (used for headers and nav background)
  • Accent Color: #e63946 (used for links, buttons, and highlights)
  • Background Color: #f8f9fa (used for body background)
  • Text Color: #212529 (used for paragraphs)

Typography

  • Headings: 'Oswald', sans-serif
  • Body Text: 'Open Sans', sans-serif
